# Changelog

## 0.1.0 — 2025-03-21

第一幕教学关可玩封板：8×8 棋盘、昼夜流程、商店与日志等核心循环已打通。

### 新增 / 玩法

- 白天警长四向移动（方向键）、每格移动约 1s 动画，动画期间锁定输入。
- 每局限一次「调查」（空格）：曼哈顿距离 2 步内扒手当日显形相关逻辑。
- 商店消耗金币购买志愿者；志愿者随机空格子生成、随机朝向。
- 夜晚扒手与志愿者自动行动；右侧扒手头像高亮当前行动者；志愿者路径拖尾等表现。
- 物品拾取获得金币；逮捕扒手、胜利/结束条件与日志输出。
- 开局警长在**随机空地**出生；日志记录坐标。

### 画面与 UI

- 警长 / 志愿者 / 扒手使用 `MapAnimatedActor`（`AnimatedSprite2D` + 32×32 四向图集），与棋盘叠放在 `SubViewport` 中。
- 右侧信息栏：回合、扒手占位、商店价签、步数与调查状态、可滚动日志；顶栏标题与版本号。
- 修正左侧地图面板纵向过度撑开导致顶栏被挤出视口的问题（面板 `size_flags`、视口与棋盘最小尺寸对齐、`HBox` 顶对齐）。
- 首帧布局未完成时角色坐标错误：`_ready` 中在生成地图按钮后 `await` 一帧再开局，避免警长出现在棋盘外。

### 工程与资源

- 运行所需角色贴图收敛到 `assets/sprites/where/`，`Main.gd` 引用已切换；完整 `character-pack` 等大包默认写入 `.gitignore` 以精简仓库。
- 移除未再使用的 `ActorSprite`（`scripts/ui/actor_sprite.gd`）。
- 完善 `.gitignore`（`.godot/`、导出、系统杂项、可选大型素材目录等）。
